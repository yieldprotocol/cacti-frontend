## Getting Started

To run the development server:

```sh
npm run dev
# or
yarn dev
# or
pnpm dev
```

If you have trouble running the above, try the below first:

```
volta pin yarn@latest
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
