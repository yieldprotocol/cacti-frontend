<p>
  CactiComponents is a modular set of User Interface (UI) components designed to
  facilitate seamless integration of new platforms and protocols into
  Cacti-Chat, an AI chatbot specifically designed to simplify decentralized
  finance (DeFi) actions and allow for easy sharing across various protocols.
  These components are aimed at creating a standardized, efficient, and
  high-quality UI, thereby streamlining the development process and ensuring an
  excellent user experience.
</p>
<p>
  Each component is a self-contained, reusable piece of code that standardizes
  certain elements of the UI. Using these components, developers can construct
  complex UIs by combining smaller, manageable pieces. This modularity not only
  promotes reusability and consistency but also improves maintainability and
  scalability of the application.
</p>

TextResponse

HeaderResponse

IconResponse

ListResponse
This component helps display list-based responses. Whether it's a list of options, a series of steps, or any other type of ordered or unordered list, ListResponse ensures a clear and organized presentation.
ActionResponse
The ActionResponse component is designed to display actionable responses, such as buttons, links, or commands. This component is crucial in guiding the user through different actions they can take in the DeFi environment.
ImageResponse
